Proses GET: Browser kirim HTTP GET → api.php baca ?action= → PHP jalankan SELECT JOIN 3 tabel → hasil dikonversi ke JSON dan dikirim.

POST Tambah Shinobi Baru

Semua field bertanda (*) wajib diisi. Elemen dan Misi bersifat opsional.

Proses POST: Form kirim HTTP POST + body JSON → api.php validasi field wajib → INSERT INTO shinobi → Response 201 Created.

PUT Update Data Shinobi

Masukkan ID yang ingin diubah, lalu isi field yang ingin diperbarui. Field kosong tidak akan diubah.

Proses PUT: Form kirim HTTP PUT + JSON → Cek shinobi ada → UPDATE dinamis (hanya kolom yang dikirim) → Response 200 OK.

DELETE Hapus Shinobi

Hapus secara permanen. Isi minimal 1 kriteria. Bisa kombinasikan beberapa kriteria sekaligus.

Proses DELETE: Form kirim HTTP DELETE + JSON kriteria → API bangun WHERE dinamis → Hitung yang cocok → DELETE → Response berisi daftar yang terhapus.

Cara Kerja REST API Ini

Arsitektur Sistem

File Fungsi Peran
koneksi.php Koneksi Database Buka koneksi MySQL via mysqli, di-include oleh api.php
setup.php Installer Buat database, 3 tabel berelasi, isi 10+ data dummy. Jalankan sekali.
api.php API Router Terima HTTP request, proses SQL, kembalikan response JSON

Alur Proses Setiap Method

GET: Client kirim HTTP GET → api.php baca ?action= → PHP jalankan SELECT JOIN 3 tabel → konversi array ke JSON → kirim response
POST: Client kirim HTTP POST + body JSON → api.php baca php://input → validasi field wajib → INSERT INTO shinobi → response 201
PUT: Client kirim HTTP PUT + body JSON (wajib id) → cek shinobi ada → bangun UPDATE dinamis (hanya kolom yang dikirim) → response 200
DELETE: Client kirim HTTP DELETE + body JSON → bangun WHERE dinamis dari kriteria → preview data yang akan dihapus → DELETE → response + daftar terhapus

3. api.php — Router CRUD Lengkap (GET / POST / PUT / DELETE)

File utama API. Mendukung 8 endpoint GET, POST (insert), PUT (update dinamis), DELETE (hapus fleksibel).

<?php
header("Content-Type: application/json");
header("Access-Control-Allow-Origin: *");
header("Access-Control-Allow-Methods: GET, POST, PUT, DELETE");
header("Access-Control-Allow-Headers: Content-Type, Authorization, X-Requested-With");

include 'koneksi.php';
$method = $_SERVER['REQUEST_METHOD'];

function sendResponse($status, $message, $data = null) {
    $r = ["meta" => ["status"=>$status, "message"=>$message, "timestamp"=>gmdate("Y-m-d\TH:i:s.v\Z")]];
    if ($data !== null) $r["data"] = $data;
    http_response_code($status);
    echo json_encode($r);
    exit;
}

switch ($method) {

  // ==================== GET ====================
  case 'GET':
    if (!isset($_GET['action'])) {
        $res = $conn->query("SELECT s.*, c.nama_clan, d.nama_desa FROM shinobi s JOIN clan c ON s.clan_id=c.id JOIN desa d ON s.desa_id=d.id");
        sendResponse(200, "Daftar Shinobi", $res->fetch_all(MYSQLI_ASSOC));
    }
    $action = $_GET['action'];

    if ($action == 'getAll') {
        $res = $conn->query("SELECT s.id, s.nama_shinobi, s.\`rank\`, s.elemen_chakra, s.jumlah_misi_selesai, c.nama_clan, c.jutsu_khas, d.nama_desa, d.lokasi_negara FROM shinobi s JOIN clan c ON s.clan_id=c.id JOIN desa d ON s.desa_id=d.id");
        $data = $res->fetch_all(MYSQLI_ASSOC);
        foreach ($data as &$row) $row['jumlah_misi_selesai'] = (int)$row['jumlah_misi_selesai'];
        sendResponse(200, "Daftar Seluruh Shinobi", $data);
    }
    elseif ($action == 'getById' && isset($_GET['id'])) {
        $id = $conn->real_escape_string($_GET['id']);
        $res = $conn->query("SELECT s.id, s.nama_shinobi, s.\`rank\`, s.elemen_chakra, s.jumlah_misi_selesai, c.nama_clan, c.jutsu_khas, d.nama_desa, d.lokasi_negara FROM shinobi s JOIN clan c ON s.clan_id=c.id JOIN desa d ON s.desa_id=d.id WHERE s.id='$id'");
        $data = $res->fetch_assoc();
        if ($data) { $data['jumlah_misi_selesai'] = (int)$data['jumlah_misi_selesai']; sendResponse(200, "Profil Shinobi", $data); }
        else sendResponse(404, "Shinobi tidak ditemukan");
    }
    elseif ($action == 'count') {
        $res = $conn->query("SELECT COUNT(*) as total FROM shinobi");
        sendResponse(200, "Total Shinobi", ["total_shinobi" => (int)$res->fetch_assoc()['total']]);
    }
    elseif ($action == 'stats') {
        $res = $conn->query("SELECT COUNT(*) as total, MAX(jumlah_misi_selesai) as max_misi, ROUND(AVG(jumlah_misi_selesai),0) as avg_misi FROM shinobi");
        $d = $res->fetch_assoc();
        sendResponse(200, "Statistik", ["total_shinobi"=>(int)$d['total'], "misi"=>["rata_rata"=>(int)$d['avg_misi'], "rekor"=>(int)$d['max_misi']]]);
    }
    elseif ($action == 'top') {
        $limit = isset($_GET['limit']) ? (int)$_GET['limit'] : 10;
        $res = $conn->query("SELECT s.id, s.nama_shinobi, s.\`rank\`, s.jumlah_misi_selesai, c.nama_clan, d.nama_desa FROM shinobi s JOIN clan c ON s.clan_id=c.id JOIN desa d ON s.desa_id=d.id ORDER BY s.jumlah_misi_selesai DESC LIMIT $limit");
        $data = $res->fetch_all(MYSQLI_ASSOC);
        foreach ($data as &$row) $row['jumlah_misi_selesai'] = (int)$row['jumlah_misi_selesai'];
        sendResponse(200, "Top $limit Shinobi", $data);
    }
    elseif ($action == 'search') {
        $where = [];
        if (isset($_GET['nama'])) $where[] = "s.nama_shinobi LIKE '%".$conn->real_escape_string($_GET['nama'])."%'";
        if (isset($_GET['clan'])) $where[] = "c.nama_clan LIKE '%".$conn->real_escape_string($_GET['clan'])."%'";
        if (isset($_GET['desa'])) $where[] = "d.nama_desa LIKE '%".$conn->real_escape_string($_GET['desa'])."%'";
        if (isset($_GET['rank'])) $where[] = "s.\`rank\` LIKE '%".$conn->real_escape_string($_GET['rank'])."%'";
        if (isset($_GET['elemen'])) $where[] = "s.elemen_chakra LIKE '%".$conn->real_escape_string($_GET['elemen'])."%'";
        $sql = "SELECT s.id, s.nama_shinobi, s.\`rank\`, s.elemen_chakra, s.jumlah_misi_selesai, c.nama_clan, d.nama_desa FROM shinobi s JOIN clan c ON s.clan_id=c.id JOIN desa d ON s.desa_id=d.id";
        if (count($where) > 0) $sql .= " WHERE ".implode(" AND ", $where);
        $data = $conn->query($sql)->fetch_all(MYSQLI_ASSOC);
        sendResponse(200, "Hasil Pencarian (".count($data)." ditemukan)", $data);
    }
    elseif ($action == 'getDesa') {
        sendResponse(200, "Daftar Desa", $conn->query("SELECT * FROM desa")->fetch_all(MYSQLI_ASSOC));
    }
    elseif ($action == 'getClan') {
        sendResponse(200, "Daftar Clan", $conn->query("SELECT c.*, d.nama_desa FROM clan c JOIN desa d ON c.desa_id=d.id")->fetch_all(MYSQLI_ASSOC));
    }
    else { sendResponse(400, "Action tidak valid"); }
    break;

  // ==================== POST ====================
  case 'POST':
    $data = json_decode(file_get_contents("php://input"), true);
    if (!$data) $data = $_POST;
    if (isset($data['id'], $data['nama_shinobi'], $data['clan_id'], $data['desa_id'], $data['rank'])) {
        $id = $conn->real_escape_string($data['id']);
        $nama = $conn->real_escape_string($data['nama_shinobi']);
        $clan_id = (int)$data['clan_id'];
        $desa_id = (int)$data['desa_id'];
        $rank = $conn->real_escape_string($data['rank']);
        $elemen = isset($data['elemen_chakra']) ? $conn->real_escape_string($data['elemen_chakra']) : '-';
        $misi = isset($data['jumlah_misi_selesai']) ? (int)$data['jumlah_misi_selesai'] : 0;
        $sql = "INSERT INTO shinobi (id, nama_shinobi, clan_id, desa_id, \`rank\`, elemen_chakra, jumlah_misi_selesai) VALUES ('$id','$nama',$clan_id,$desa_id,'$rank','$elemen',$misi)";
        if ($conn->query($sql)) sendResponse(201, "Shinobi berhasil didaftarkan", ["id"=>$id, "nama_shinobi"=>$nama]);
        else sendResponse(500, "Database Error: ".$conn->error);
    } else {
        sendResponse(400, "Field wajib: id, nama_shinobi, clan_id, desa_id, rank");
    }
    break;

  // ==================== PUT ====================
  case 'PUT':
    $data = json_decode(file_get_contents("php://input"), true);
    if (!isset($data['id'])) sendResponse(400, "Field 'id' wajib dikirim");
    $id = $conn->real_escape_string($data['id']);
    $check = $conn->query("SELECT id FROM shinobi WHERE id='$id'");
    if ($check->num_rows == 0) sendResponse(404, "Shinobi '$id' tidak ditemukan");
    $fields = [];
    if (isset($data['nama_shinobi'])) $fields[] = "nama_shinobi='".$conn->real_escape_string($data['nama_shinobi'])."'";
    if (isset($data['clan_id'])) $fields[] = "clan_id=".(int)$data['clan_id'];
    if (isset($data['desa_id'])) $fields[] = "desa_id=".(int)$data['desa_id'];
    if (isset($data['rank'])) $fields[] = "\`rank\`='".$conn->real_escape_string($data['rank'])."'";
    if (isset($data['elemen_chakra'])) $fields[] = "elemen_chakra='".$conn->real_escape_string($data['elemen_chakra'])."'";
    if (isset($data['jumlah_misi_selesai'])) $fields[] = "jumlah_misi_selesai=".(int)$data['jumlah_misi_selesai'];
    if (count($fields) == 0) sendResponse(400, "Tidak ada field untuk di-update");
    if ($conn->query("UPDATE shinobi SET ".implode(",",$fields)." WHERE id='$id'"))
        sendResponse(200, "Shinobi '$id' berhasil diperbarui");
    else sendResponse(500, "Database Error: ".$conn->error);
    break;

  // ==================== DELETE ====================
  case 'DELETE':
    $data = json_decode(file_get_contents("php://input"), true);
    if (!$data) $data = [];
    if (isset($_GET['id'])) $data['id'] = $_GET['id'];
    $where = [];
    if (!empty($data['id'])) $where[] = "s.id='".$conn->real_escape_string($data['id'])."'";
    if (!empty($data['nama_shinobi'])) $where[] = "s.nama_shinobi LIKE '%".$conn->real_escape_string($data['nama_shinobi'])."%'";
    if (!empty($data['clan'])) {
        $v = $conn->real_escape_string($data['clan']);
        $cr = $conn->query("SELECT id FROM clan WHERE nama_clan LIKE '%$v%'");
        $cids = []; while ($row=$cr->fetch_assoc()) $cids[]=(int)$row['id'];
        if (count($cids)>0) $where[] = "s.clan_id IN (".implode(',',$cids).")";
        else sendResponse(404, "Clan '$v' tidak ditemukan");
    }
    if (!empty($data['desa'])) {
        $v = $conn->real_escape_string($data['desa']);
        $dr = $conn->query("SELECT id FROM desa WHERE nama_desa LIKE '%$v%'");
        $dids = []; while ($row=$dr->fetch_assoc()) $dids[]=(int)$row['id'];
        if (count($dids)>0) $where[] = "s.desa_id IN (".implode(',',$dids).")";
        else sendResponse(404, "Desa '$v' tidak ditemukan");
    }
    if (!empty($data['rank'])) $where[] = "s.\`rank\`='".$conn->real_escape_string($data['rank'])."'";
    if (!empty($data['elemen_chakra'])) $where[] = "s.elemen_chakra LIKE '%".$conn->real_escape_string($data['elemen_chakra'])."%'";
    if (count($where)==0) sendResponse(400, "Minimal 1 kriteria harus dikirim");
    $w = implode(" AND ", $where);
    $cnt = $conn->query("SELECT COUNT(*) as t FROM shinobi s WHERE $w")->fetch_assoc()['t'];
    if ((int)$cnt == 0) sendResponse(404, "Tidak ada shinobi yang cocok");
    $prev = $conn->query("SELECT s.id, s.nama_shinobi FROM shinobi s WHERE $w")->fetch_all(MYSQLI_ASSOC);
    if ($conn->query("DELETE s FROM shinobi s WHERE $w"))
        sendResponse(200, "$cnt shinobi berhasil dihapus", ["total_dihapus"=>(int)$cnt, "data_terhapus"=>$prev]);
    else sendResponse(500, "Database Error: ".$conn->error);
    break;

  default:
    sendResponse(405, "Method tidak diizinkan. Gunakan: GET, POST, PUT, DELETE");
}
$conn->close();
?>

4. Tunggu akun aktif (1-5 menit)

Langkah 2 - Buat Database MySQL

1. Di dashboard klik MySQL Databases
2. Isi nama database → klik Create Database
3. Catat: Database Host, Database Name, Username, Password
4. Buka phpMyAdmin → pilih database → tab SQL → paste query CREATE TABLE dari setup.php

Langkah 3 - Edit koneksi.php

Ganti isi koneksi.php sesuai data dari panel InfinityFree:

<?php
$host = "sql123.infinityfree.com";  // dari panel MySQL
$user = "if0_xxxxxxxx";             // username database
$pass = "password_kamu";            // password database
$db   = "if0_xxxxxxxx_naruto_api";  // nama lengkap database
$conn = new mysqli($host, $user, $pass, $db);
if ($conn->connect_error) {
    die(json_encode(["error" => $conn->connect_error]));
}
?>

Langkah 4 - Upload via FileZilla

1. Buka FileZilla
2. Info FTP dari panel InfinityFree → FTP Accounts:
  • Host: ftpupload.net
  • Username dan Password sesuai panel
  • Port: 21
3. Klik Quickconnect
4. Navigasi ke folder htdocs/ di panel kanan
5. Upload: index.html, api.php, koneksi.php
6. Tunggu selesai

Langkah 5 - Uji API Online

1. Buka phpMyAdmin → paste query INSERT dari setup.php untuk isi data
2. Test di browser: https://namadomain.rf.gd/api.php?action=getAll
3. Test di Postman: ganti localhost:8000 dengan domain InfinityFree
4. Test halaman web: https://namadomain.rf.gd/index.html

Troubleshooting

Masalah Solusi
Error koneksi database Cek hostname, username, password di panel MySQL InfinityFree
PUT/DELETE tidak bekerja InfinityFree shared hosting kadang blokir - coba via Postman langsung
File tidak muncul Pastikan upload ke folder htdocs/ bukan root
CORS error Sudah ada header Access-Control-Allow-Origin: * di api.php
Tips: Setelah hosting online, ubah URL di konstanta API di bagian script dari localhost:8000 ke domain InfinityFree agar form interaktif langsung terhubung ke server online.
